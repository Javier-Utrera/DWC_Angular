Sobre este proyecto

Esta aplicación te permite explorar y descubrir libros de la API de Google Books. Usa el buscador para encontrar libros por título, autor o categoría.

Ademas he usado dos proveedores externos en mi proyecto, Firebase para el login y la base de datos y emailjs para la mensajeria

Tambien he incluido otra api proporcionada por google books difente a la que hago para las busquedas, esta me sirve para visualizar vistras previas del libro en el componente libro-detalle
